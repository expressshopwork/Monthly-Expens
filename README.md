# 💰 Monthly-Expens — តាមដានចំណាយគ្រួសារ / Family Expense Tracker

កម្មវិធីតាមដានការចំណាយប្រចាំខែសម្រាប់គ្រួសារ បង្កើតដោយ HTML, CSS, និង JavaScript ។

A monthly expense tracking app for families, built with HTML, CSS, and JavaScript.

---

## ✨ លក្ខណៈពិសេស / Features

- 👨👩 តាមដានចំណូល/ចំណាយ បែងចែកតាមប្ដី និងប្រពន្ធ / Track income & expenses per member (Husband / Wife)
- 📊 តារាងប្រៀបធៀបចំណូល, ចំណាយ, និងសមតុល្យសុទ្ធ / Comparison table: income, expense, net balance per member
- 📊 Progress Bar បង្ហាញការចំណាយប្ដី VS ប្រពន្ធ / VS progress bar showing expense split
- 🗂️ ប្រអប់ Tab ត្រងទិន្នន័យ: ទាំងអស់ / ប្ដី / ប្រពន្ធ / Filter tabs: All / Husband / Wife
- 💾 រក្សាទុកទិន្នន័យក្នុង LocalStorage / Data persisted in LocalStorage
- 📱 ប្រើប្រាស់បានល្អលើទូរសព្ទ / Mobile-responsive design
- 🗑️ លុបធាតុម្តីៗ ឬលុបតាម Member / Delete individual transactions or clear by member
- 💱 ជ្រើសរើសប្រភេទ: ចំណូល ឬ ចំណាយ / Select type: Income or Expense
- 🗂️ ជ្រើសរើសប្រភេទទំនិញ / Category selection with icons

---

## 🚀 របៀបប្រើ / Usage

1. បើក `index.html` ក្នុង Browser ។ / Open `index.html` in your browser.
2. ជ្រើសរើសសមាជិក (ប្ដី ឬ ប្រពន្ធ) ។ / Select a member (Husband or Wife).
3. ជ្រើសរើសប្រភេទ (ចំណូល ឬ ចំណាយ) ។ / Select the type (Income or Expense).
4. បំពេញពិពណ៌នា, ចំនួនទឹកប្រាក់, និងប្រភេទ ។ / Fill in description, amount, and category.
5. ចុច **💾 រក្សាទុក** ដើម្បីបន្ថែម ។ / Click **💾 Save** to add the entry.
6. ប្រើ Tab ខាងលើដើម្បីត្រងទិន្នន័យ ។ / Use the tabs at the top to filter by member.
